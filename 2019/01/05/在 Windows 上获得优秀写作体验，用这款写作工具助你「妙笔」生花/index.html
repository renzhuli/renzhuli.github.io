Markdown 是许多人的日常需求，在 macOS 上，我们有 Ulysses、Bear 这样的优质 Markdown 编辑器，但在 Windows 平台上，却鲜有这样优秀的编辑器。今天要介绍的妙笔，便是支持 Windows 与 macOS 双平台的一款优质 Markdown 编辑器。

妙笔最突出的地方，便是它可以让我们更加专注于自己正在撰写的文章上，让我们能够更高效率地完成我们的作品。

文档目录

在撰写长篇文章时，条理性是非常重要的。妙笔采用了树状目录，让我们在写作时能够保持更好的条理性。当目录树嵌套较复杂的情况下，我们也可以点击工具栏的「切换聚焦」按钮，聚焦于正在处理的文档。此时，无关的部分都会被隐藏。

得益于清晰的目录，我们也可以直接在目录处拟写大纲，再逐个完善。完成写作后，我们也可以将一个目录节点与它的所有子节点一次性导出。

实时存档与备份

如果你跟我一样，曾因为忘记保存而丢过稿，相信你一定懂得随手保存的重要性。在妙笔中，自动实时保存的功能则解放了我们频繁按下 Ctrl+S 的手。我们可以在妙笔的选项窗口中打开「文档历史记录」，并设置文档自动保存的时间。这样一来，只要在这个时间内，我们的文章有过任何修改，它都会被保存下来。

自动保存

我们可以在侧边栏的「文档快照」中查看到自动保存的快照。我们可以将当前的文本与历史版本进行对比，妙笔会为我们标记出两者的不同，在修改文章时，让我们对文章内容能有更好的把控。

文档快照

除了单个文档的自动保存外，妙笔也可以对整个文档库进行自动备份。我们可以在选项窗口的「备份」面板中开启自动备份功能，每隔 15 分钟，妙笔便会对整个文档库进行备份。即使你不小心删除了某个文档，你也可以随时恢复到之前的备份，避免丢失重要内容。

打造你的专属编辑器

我们每个人打开妙笔的理由都不尽相同，也许是撰写一篇文章，也许只是写一篇五分钟日记。无论你打开的原因是什么，想必你都希望可以获得美妙的写作体验。妙笔支持对编辑器背景进行自定义，你可以根据心情来更换自己喜欢的背景，让自己写作时的心情更加舒畅。

自定义背景

此外，我们也可以对编辑器的宽度、行间距乃至缩进进行调整。打造一款让自己用起来更舒服的编辑器。

调整宽度、间距等

如果你打开妙笔是为了写五分钟日记，或者你经常需要对模板化的内容进行修改。那么妙笔的「文档模板」功能一定会让你惊呼过瘾。在选项窗口的「高级」面板中，我们可以创建不同的文档模板。当我们新建文档时，得到的不再是空白的文档，而是预先设置好的模板内容。这样一来，我们只需要将内容填写进模板即可。

自定义模板

此外，我们也可以在「高级」面板中自定义样式和脚本。在我们预览或导出为 HTML 文件时，便可以直接应用预设好的 CSS 样式和 JavaScript 脚本。为了安全起见，自定义脚本默认只在导出的 HTML 中生效，预览则不生效，你也可以手动修改这一设置。

文章备注

文章并非一蹴而就的，无论是撰写长篇小说，还是应用测评，我们都需要用到备注功能来记下一些重要的内容。从而在撰写或修改时，避免偏离主题。妙笔提供了文档备注与全局备注两种备注模式，帮助我们记录灵感，收集资料。

文档备注在右侧的边栏中，我们在写作过程中可以随时查阅。一般而言，文档备注都是用来记录一些灵感，或是撰写当前文档的大纲。因此，文档备注正如其名，只能在当前文档中查看。

文档备注

全局备注则是面向整个文档库的备注面板。全局备注采用了类似 Trello 和 Teambition 那样的卡片形式。我们可以在看板中查看每一张卡片，从而让整体逻辑更加清晰。如果你正在撰写一篇长篇小说，不妨使用全局备注来记录各种细节设定。

全局备注

除了通过编辑器右上角的按钮进入全局备注的看板外，我们还能通过 Ctrl（macOS 则为 Command）+4 的快捷键来进入看板。

窗口切分

在日常写作时，我们时常会需要打开其他文档进行参考。比如在翻译一篇外语文章时，我们需要有一个地方显示原文。新开一个窗口又会挤占过多屏幕空间，导致编辑界面过于拥挤。此时，不妨试试妙笔的「窗口切分」功能。

窗口切分

我们可以通过快捷键 Ctrl/Command + Shift + \ 或是点击编辑界面右下角的按钮来切分窗口。在窗口切分后，活跃窗口的左上角会有一个绿色的圆点作为提示，此时你所输入的内容都会被添加到活跃窗口的文档中。

更多功能

除了上述提到的功能外，妙笔作为一款优秀的编辑器，同时也支持这些功能：

  • 多种导出格式：我们可以用妙笔直接将文章导出为 PNG 图片、PDF、DOCX 文档、HTML 等多种格式。
  • 全屏专注模式：我们可以通过顶部的「专注模式」按钮来进入全屏专注模式，将无关内容隐藏。
  • 数据安全：妙笔的所有数据都是保存在本地的，因此我们不用担心由于服务器故障带来的文档丢失或泄露。如果你需要在多台设备上工作，你也可以将文档库设立在 iCloud 或 Dropbox 中进行同步。
  • 快捷按键：妙笔提供了不少好用的快捷按键，我们可以通过 Ctrl/Command + J 来快速跳转到其他文档，或是通过 Ctrl/Command + Shift + L 来锁定界面等。在妙笔的菜单栏中，我们可以看到各种操作所对应的快捷键。

写在后面

在如今优质的 Markdown 编辑器大多只提供 macOS 或 iOS 版本时，作为一款优秀的编辑器，妙笔为 Windows 平台的用户带来了极佳的 Markdown 体验。无论你是需要撰写大量文章的专业作者，还是需要写作的学生，又或者只是一个写作爱好者，妙笔都是一个不错的选择。
